Before you start using the bot, follow these steps:
1. install python from https://python.org/ (if you don't have already)
2. download main.py and members.txt to your computer (files must be in one folder)
3. open main.py in any text editor
4. go to https://discord.com/developers/applications/ and login with your discord account.
5. create new application or go to the existing one.
6. go to the bot section and click reset token
7. copy this token.
8. go back to your text editor and go to the last line.
9. replace YOUR TOKEN with copied token.
10. save the file.
11. open the windows explorer and go to bot's folder.
12. type cmd in textbox at the top.
13. in cmd type python main.py
14. done!
